Dog Playlist

About

This website generates a playlist where each song is based off of a letter in a random dog breed chosen from the Dog API. The songs are found using the Spotify API. Many songs have a playable snippet, but if the play and pause buttons are missing, the Spotify API was missing a preview of the song.
