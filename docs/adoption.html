Caltech Critters Adoption Agency

About

Please view the final product at Caltech Critters Adoption Website. Source code for this project can be found on my GitHub page.

This project was created with a custom API and retro-themed interactive website. The API has six GET and eight POST endpoints. Some features include updating the API upon purchase, filtering animals by their species, and logging in as an administrator to add more available animals. The inspiration for the project is Caltech's own wildlife! Every picture is taken by myself or friends who attend Caltech.
